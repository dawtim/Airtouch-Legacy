# AirTouch Legacy for Home Assistant

HACS-ready custom integration scaffold for legacy Polyaire AirTouch controllers.

## Features
- Config flow
- Climate entity
- Zone switch entities
- Zone temperature sensors
- Zone damper number entities
- Diagnostics support

## Important
This package is a reverse-engineered starting point.
Polling of status is grounded in packet captures.
Write operations and field offsets may need adjustment on your exact controller.
